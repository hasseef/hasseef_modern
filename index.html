القطاع الحكومي / إمارة المنطقة

  • إدارة الفعاليات: إضافة/تسجيل/ربط مرافق ومتحدثين/تقييم.
  • إدارة المتحدثين: استقبال الطلبات، المسح الأمني، الاعتماد.
  • إدارة المرافق: إتاحة مرافق حكومية وحجزها، تقارير الاستخدام.
  • إدارة المشاريع التنموية وربطها برؤية 2030.
  • حلول التنمية: طلب/تقييم/تنفيذ الحلول.
  • التدريب التعاوني، التوظيف، التطوع.
  • تقارير رؤية 2030، الشراكات، المحفظة.

هيئة التطوير (Development Authority)

  • مراجعة واعتماد: فعاليات/متحدثين/حجوزات مرافق.
  • تقارير وارتباطات رؤية 2030 على مستوى المنطقة/المدينة.
  • حلول التنمية: مراجعة وانتقاء الحلول وتحويلها لمشاريع.
  • الشراكات الاستراتيجية وإدارة الأثر.

القطاع الخاص

  • المسؤولية الاجتماعية: تصميم مبادرات وربطها بالرؤية.
  • المشاريع والتنمية: تمويل مشترك/تقارير الأثر.
  • فرص الاستثمار والاكتتابات.
  • التوظيف + برامج التدريب + إدارة المرافق.
  • الشراكات والحلول، التبرعات والمحفظة.

القطاع غير الربحي

  • إدارة المشاريع والمبادرات + طلب الاستشارات العلمية.
  • الحلول: طلب/تقييم/تنفيذ مع الجامعات.
  • التدريب/التوظيف/التطوع.
  • الفعاليات/المتحدثون/المرافق.
  • التبرعات والمحفظة والشراكات.

الجامعة

  • الاستشارات العلمية: استقبال/تقييم/رسوم إدارية/متابعة أثر.
  • الأبحاث وربطها بالمشاريع/الحلول.
  • التدريب التعاوني، الفعاليات، المتحدثون.
  • التطوع، الشراكات، المحفظة والمنح.
  • تقارير رؤية 2030 (الدعم العلمي).

الأفراد

  • وظائف/تدريب/تطوع + فحص الميول وتوصيات ذكية.
  • إطلاق مبادرات فردية وربطها بالرؤية والجهات.
  • محفظة/تبرعات/استثمارات صغيرة.
  • التحدث/التدريب وفق المؤهلات.

المدربون

  • إدارة دورات (مدفوعة/تطوعية) + الحقائب التدريبية.
  • طلبات تدريب من الجهات + تقارير الحضور والشهادات.
  • الفعاليات، الشراكات، التقييمات.

حاضنات ومسرّعات الأعمال

  • دعم الشركات الناشئة/توجيه/تسويق/شراكات.
  • محفظة استثمارية/تقارير الأداء.
  • طرح فرص/حلول للمشاريع التنموية.

الجهات المانحة/المموّلون

  • فرص منح بحسب المعايير/محفظة مانح.
  • تقارير الأثر والشفافية/الرعايات.
  • التكامل مع المشاريع عبر الرؤية.

المكونات المشتركة (واجهات وخدمات)

قابلة للتوسعة
المشاريع والمبادرات
  • CRUD المشاريع + مراحل التنفيذ + الميزانية + المرفقات + الربط برؤية 2030.
  • التمويل المشترك/المنح/التبرعات + تقارير الأثر.
  • مسارات موافقات متعددة المراحل (قابلة للضبط).
المحافل والفعاليات
  • إنشاء فعالية/الجمهور/الجدولة/التذاكر/شهادات الحضور.
  • ربط متحدثين معتمدين ومرافق محجوزة.
  • مراجعات واعتمادات (حكومة/هيئة تطوير).
الفرص (وظائف/تدريب/تطوع)
  • محرك مطابقة يعتمد الميول والمؤهلات والخبرة.
  • تقارير التقدم، رسائل إشعار، لوحات مرشح/جهة.
المحفظة الرقمية والتعاملات
  • محافظ حسب الحساب + حركات مالية (إيداع/سحب/تبرع/استثمار).
  • تحصيل رسوم/رعايات/إعلانات/نسب مشاريع.
المرافق والحجوزات
  • إضافة مرافق/سعة/سعر/صور/شروط.
  • حجز بمراجعة واعتماد + تقارير استخدام.
المتحدثون
  • سجل متحدثين: طلب/مسح/اعتماد/تصنيف.
  • ربط بالفعاليات + تقييمات.
الحلول
  • طرح تحديات + استقبال حلول من القطاعات/الأفراد/الجامعات.
  • لجان تقييم + تحويل الحلول لمشاريع تنفيذية.
الشراكات
  • اقتراح/تفاوض/تتبع/تقارير أثر.
الاستشارات العلمية والأبحاث والتدريب التعاوني
  • طلبات استشارة للجامعات + رسوم إدارية + متابعة أثر.
  • ربط الأبحاث بالمشاريع/الحلول + نشر وتسويق نتائج.
  • برامج تدريب تعاوني بإدارة الطلبات والتقييمات.

نماذج البيانات (مُلخّص)

قابل للتطبيق على أي Back‑End

Users

  • id, fullName, email, role, walletId
  • الأدوار: INDIVIDUAL, PRIVATE, NONPROFIT, GOVERNMENT, UNIVERSITY, DONOR, TRAINER, INCUBATOR, DEVELOPMENT_AUTHORITY

Projects / Events / Opportunities

  • Project: title, description, budget, status, ownerId
  • Event: title, date, location, ownerId
  • Opportunity: kind (JOB/TRAINING/VOLUNTEER), title...

Wallet / Donation

  • Wallet: balance
  • WalletTxn: amount, kind (DEPOSIT/WITHDRAW/DONATION/INVESTMENT)
  • Donation: donorId, projectId, amount

Facilities / Bookings

  • Facility: name, location, capacity, price, ownerId
  • FacilityBooking: facilityId, requesterId, dateFrom..To, status

Speakers

  • Speaker: fullName, bio, field, status

Solutions / Partnerships

  • Solution: title, description, submittedById, status
  • Partnership: title, parties(json), status, ownerId

VisionGoal / VisionLink

  • VisionGoal: code, title, level
  • VisionLink: goalId, entity, entityId
